Welcome to your shiny new codespace! We've got everything fired up and running for you to explore Python and Jupyter notebooks.

## 🤝 แชร์ Codespace ให้เพื่อน (Sharing Your Codespace)

คุณสามารถแชร์ Codespace ให้เพื่อนร่วมทำงานได้แบบ Real-time โดยใช้ **VS Code Live Share** ที่ติดตั้งมาพร้อมกับ Codespace นี้แล้ว

**วิธีใช้งาน:**

1. เปิด Command Palette (`Ctrl+Shift+P` หรือ `Cmd+Shift+P` บน Mac)
2. พิมพ์ `Live Share: Start Collaboration Session` แล้วกด Enter
3. ลงชื่อเข้าใช้ด้วย GitHub account ของคุณ (ครั้งแรกเท่านั้น)
4. ลิงก์ invitation จะถูกคัดลอกไปยัง clipboard อัตโนมัติ
5. ส่งลิงก์นั้นให้เพื่อน — เพื่อนคลิกลิงก์เพื่อเข้าร่วม session ได้เลย

**You can share your Codespace with friends for real-time collaboration using the built-in VS Code Live Share extension:**

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Type `Live Share: Start Collaboration Session` and press Enter
3. Sign in with your GitHub account (first time only)
4. An invitation link is automatically copied to your clipboard
5. Send the link to your friend — they click it to join your session
